# energy-markets-python


Python tools for natural gas market analysis. Built to develop 
quantitative skills for energy trading and commodities roles.

---

## Projects

### 1. Henry Hub Price Analyzer
`price_analyzer.py` — Pure Python (no libraries). Computes price 
stats and market tone from a series of Henry Hub spot prices.

### 2. EIA Storage vs Seasonal Average
`storage_chart.py` — Pulls live weekly storage data from the EIA 
API and plots it against the 5-year min/max range.
